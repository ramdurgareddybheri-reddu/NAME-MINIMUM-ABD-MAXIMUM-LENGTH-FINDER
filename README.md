# NAME-MINIMUM-AND-MAXIMUM-LENGTH-FINDER
🔠 Minimum and Maximum Name Length Finder
A simple Python utility that takes a list of names and identifies the shortest and longest names based on character length.

📖 Overview
This project helps users quickly analyze a list of names by:

Finding the name with the minimum length.

Finding the name with the maximum length.

Displaying both names along with their lengths.

⚙️ Features
📝 Input a list of names manually or from a file.

📊 Output shortest and longest names with lengths.

🔄 Works with any number of names.

🖥️ Lightweight and beginner-friendly script.

🚀 Installation
Clone the repository:

bash
git clone https://github.com/yourusername/name-length-finder.git
cd name-length-finder
Install dependencies (if any):

bash
pip install -r requirements.txt
▶️ Usage
Run the script:

bash
python name_length_finder.py
Example:

bash
Enter names separated by commas: Ram, Sita, Lakshmana, Bharata
Shortest name: Ram (3 characters)
Longest name: Lakshmana (9 characters)
